SimpleWebConf 2021

A conference celebrating web development made simple, from side project to enterprise!

No Frameworks

When you learn a framework, you learn a skill that doesn’t directly transfer. You learn a temporary a skill abstraction instead. Spend your time learning a language and benefit from everything browser support offers. And browsers support so many things these days!

No delays

Web Development is about running code on other computers but it can also give extremely direct feedback. The short feedback loop makes it both fun and easy to learn because you can see what you do!

No limitations

The web is amazing. There are so many things you can in the browser today. We might have lost our good friends %3Cmarque%3E, %3Cblink%3E and spacer GIFs but we get so much in return! You can do anything as a developer

Do what you like

The old school webmasters no longer exist. The web is wide and diverse, and so are the roles in web development. This is great, because you can focus on what you like! If you can imagine it, odds are, somebody else has made it (and shared it). In the end, the web is about what makes it fun for you. Part of the fun is that you can use whatever you like (including frameworks) because it’s the web! You can do whatever!
